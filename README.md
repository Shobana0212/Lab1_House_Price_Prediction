## 📌 Overview

This project demonstrates **House Price Prediction** using multiple Machine Learning regression algorithms. The notebook performs complete data analysis, preprocessing, model building, evaluation, hyperparameter tuning, and visualization on two benchmark datasets:

- California Housing Dataset
- Ames Housing Dataset

The objective is to compare different regression algorithms and identify the best-performing model for predicting house prices.

---

## 📂 Datasets

### California Housing Dataset
- Source: Scikit-learn
- 20,640 instances
- 8 numerical features
- Target: Median House Value

### Ames Housing Dataset
- Source: OpenML
- Residential property dataset
- Target: SalePrice
